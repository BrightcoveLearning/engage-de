Bearbeiten von Videoeigenschaften mithilfe der Schnellbearbeitung

In diesem Thema erfahren Sie, wie Sie mit der Schnellbearbeitungsfunktion die Videoeigenschaften bearbeiten.
 

Die Schnellbearbeitungsfunktion bietet eine bequeme Möglichkeit, Videosigenschaften für mehrere Videos zu bearbeiten. Sie können mehrere Videos auswählen, indem Sie auf ein Kontrollkästchen klicken und dann bei gedrückter Umschalttaste auf ein anderes Kontrollkästchen klicken (halten Sie die Umschalttaste gedrückt, während Sie auf das zweite Video klicken, und die beiden Videos und alle dazwischen liegenden Videos werden ausgewählt). Klicken Sie auf Schnelle Bearbeitung öffnet das Schnellbearbeitungsfenster auf der rechten Seite der Videoseite.

Im Schnellbearbeitungsfenster wird eine Teilmenge der Videoeigenschaften angezeigt:

  • Nennen
  • Kurze Beschreibung
  • Lange Beschreibung
  • Tags zum Hinzufügen
  • Zu entfernende Tags
  • Verwandte Link-URL
  • Zugehöriger Link-Text
  • Planen von Start- / Enddaten
  • Benutzerdefinierte Felder
Hinweis: Eine Beschreibung all dieser Eigenschaften finden Sie im Bearbeiten von Videoeigenschaften dokumentieren.

Verwenden Sie das Schnellbearbeitungsfenster, um Werte zuzuweisen. Wenn Sie fertig sind, klicken Sie auf speichern um die Änderungen zu speichern oder Stornieren beenden.

Hinweise zur Verwendung von Quick Edit

  • Wenn mindestens einem Video Werte zugewiesen sind, die einer Eigenschaft zugewiesen sind, Gemischten Wert bearbeiten wird Angezeigt werden
  • Ein Sternchen (*) nach dem Eigenschaftsnamen zeigt eine erforderliche Eigenschaft an
  • Klicken Sie auf, um an einer Eigenschaft vorgenommene Änderungen abzubrechen Wiederherstellen
