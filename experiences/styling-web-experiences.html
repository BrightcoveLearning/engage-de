Anpassen des Stils eines Weberlebnisses

In diesem Thema erfahren Sie, wie Sie den Stil einer Web Experience anpassen.

Mit dem Experience Editor können Sie den Stil einer Web Experience anpassen, indem Sie das Thema, die Farben und die Schriftarten ändern. Um den Stil zu bearbeiten, bearbeiten Sie die Erfahrung und klicken Sie dann auf STIL in der linken Navigation.

Auswahl eines Themas

Auswählen von a Thema Wendet einen vordefinierten Satz von Schriftarten und Farben auf das Erlebnis an. So wählen Sie a Thema::

  1. Klicken Sie im linken Navigationsbereich auf STYLE.
  2. Wählen Sie ein Thema aus der Dropdown-Liste aus. Wenn Sie ein Thema auswählen, wird die Erfahrung aktualisiert, sodass Sie eine Vorschau des Themas anzeigen können.
  3. Um zum Experience Editor zurückzukehren, klicken Sie auf < Zurück zu Experience.

Farben anpassen

Die Farbeinstellungen können verwendet werden, um die Farben, die von einem Thema bereitgestellt werden, fein abzustimmen. Zum Beispiel können Farbeinstellungen verwendet werden, um die Hintergrund- und Schriftfarben fein abzustimmen. So passen Sie die Farben an:

  1. Klicken Sie im linken Navigationsbereich auf das STIL-Symbol. Das Farben Im Abschnitt werden alle Seitenobjekte und ihre Farben angezeigt.
  2. Um eine neue Farbe auszuwählen, klicken Sie auf den Farbkreis.
  3. Wählen Sie mit der Farbauswahl eine Farbe aus, indem Sie auf die Farbe im Farbquadrat klicken oder eine hexadezimale Farbzahl unter dem Farbquadrat eingeben. Beachten Sie, dass Sie den Farbbereich ändern können, indem Sie in das Farbspektrum unter dem Farbquadrat klicken. Klicken Sie auf X, um die Farbauswahl zu schließen und die Änderung zu speichern.

    Um eine Farbauswahl zurückzusetzen, klicken Sie auf den Pfeil nach links neben dem Farbkreis.

  4. Um zum Experience Editor zurückzukehren, klicken Sie auf < Zurück zu Experience.

Schriftarten auswählen

So wählen Sie verschiedene Schriftarten für das Erlebnis aus:

  1. Klicken Sie im linken Navigationsbereich auf STYLE.
  2. Neu auswählen Schriftarten Verwenden der Dropdown-Listen.

    Klicken Sie auf, um eine Schriftartenauswahl zurückzusetzen zurücksetzen Link über der Schriftart-Dropdown-Liste.

  3. Um zum Experience Editor zurückzukehren, klicken Sie auf < Zurück zu Experience.
