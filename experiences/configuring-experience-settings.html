Konfigurieren von Erfahrungseinstellungen

In diesem Thema erfahren Sie, wie Sie Erlebniseinstellungen konfigurieren.

Um die Experience-Einstellungen zu konfigurieren, öffnen Sie die Seite Projekte und klicken Sie oben auf der Seite auf den Link Einstellungen. Die folgenden Einstellungen können konfiguriert werden:

  • Zugriffskontrollprofile

Zugriffssteuerungsprofile konfigurieren

Zugriffssteuerungsprofile können erstellt werden, um den Zugriff auf veröffentlichte Erlebnisse zu steuern. Der Zugriff auf eine Website kann eingeschränkt werden durch:

  • SSO-Anmeldeinformationen - Für den Zugriff ist ein Benutzername ein Kennwort erforderlich

Informationen zum Erstellen von Zugriffssteuerungsprofilen finden Sie unter Erstellen von Zugriffssteuerungsprofilen.
