Videos zu einem Erlebnis hinzufügen

In diesem Thema erfahren Sie, wie Sie Videos zu einem Erlebnis hinzufügen können.

Mit dem Collection Editor können Sie Sammlungen erstellen und die Videos auswählen, die bei einem Erlebnis angezeigt werden sollen. Mit Erfahrungen werden Videos in Sammlungen unterteilt.

Erstellen einer Sammlung

Gehen Sie folgendermaßen vor, um eine Sammlung zu erstellen:

  1. Klicken VIDEOS in der linken Navigation. Die Seite „Sammlungen verwalten“ wird geöffnet. Auf der rechten Seite wird eine Vorschau des Videolayouts für die Startseite für den ausgewählten Vorlagentyp angezeigt.
  2. Geben Sie einen Namen für die Sammlung ein und drücken Sie die Eingabetaste. Der Name wird den Zuschauern angezeigt. Verwenden Sie daher unbedingt einen aussagekräftigen Namen. Klicken Sie auf das Bearbeiten () -Symbol, um den Namen der Sammlung zu bearbeiten.

Arbeiten mit mobilen App-Erlebnissen

Beim Erstellen von mobilen App-Erlebnissen wird das erste Video in der ersten Sammlung als vorgestelltes Video verwendet. Das vorgestellte Video wird oben in der Anwendung angezeigt.

Beim Erstellen von Sammlungen für mobile App-Erlebnisse empfiehlt Brightcove, eine Featured Video-Sammlung als erste Sammlung zu erstellen. Durch Zuweisen einer manuellen Wiedergabeliste zur Sammlung Featured Video kann das vorgestellte Video einfach geändert werden, indem die manuelle Wiedergabeliste über die Videoseite bearbeitet wird. Auf diese Weise können Sie auch Änderungen am vorgestellten Video vornehmen, ohne die Anwendung bearbeiten und erneut veröffentlichen zu müssen.

Eine Sammlung bearbeiten

  1. Um eine Sammlung zu bearbeiten, klicken Sie auf den Namen der Sammlung.
  2. Die Seite „Sammlung bearbeiten“ wird geöffnet. Die Seite „Sammlung bearbeiten“ wird verwendet, um Videos zu einer Sammlung hinzuzufügen und zusätzliche Sammlungseigenschaften zu konfigurieren. Die Sammlung ändern Dropdown listet alle Sammlungen auf, die erstellt wurden. Wählen Sie die zu bearbeitende Sammlung aus.
  3. Die Sammlungsquelle gibt die Quelle des Videoinhalts für die Sammlung an:
    • Brightcove - Videos von einem Brightcove-Konto werden verwendet
  4. Mit einer Brightcove Collection Source können die Videos aus einer Wiedergabeliste stammen oder eine benutzerdefinierte Zuweisung verwenden:
    • Playlist - Videos in der Sammlung basieren auf einer Playlist
    • Benutzerdefiniert - Videos in der Sammlung basieren auf Video-Tags, einer benutzerdefinierten Suche oder benutzerdefinierten Metadatenfeldern
    Hinweis: Wenn Sie kürzlich Änderungen an Video-Metadaten oder Wiedergabelisten vorgenommen haben, können diese Änderungen einige Minuten dauern, bis diese Änderungen im Collection Editor und bei veröffentlichten Erlebnissen angezeigt werden.

    Verwenden einer Playlist-Videozuweisung

    Wenn Sie eine Playlist-Videozuweisung verwenden möchten, basieren die Videos in der Sammlung auf einer Wiedergabeliste.

    1. Wählen Sie die Option Playlist aus. Eine Liste der Wiedergabelisten in deinem Account wird angezeigt.
    2. Verwenden Sie die Navigationssteuerelemente () oder die Suchfunktion, um zur gewünschten Wiedergabeliste zu suchen. Nachdem Sie eine Wiedergabeliste ausgewählt haben, wird die Liste der Videos in dieser Wiedergabeliste angezeigt.
      Hinweis: Die Reihenfolge der Videos in der Sammlung basiert auf den Einstellungen der Wiedergabeliste.
    3. Klicken Sie auf Speichern, um Ihre Änderungen zu speichern.

    Verwenden einer benutzerdefinierten Videozuweisung

    Wenn Sie sich für eine benutzerdefinierte Videozuweisung entscheiden, basieren die Videos in der Sammlung auf Video-Tags, einer benutzerdefinierten Suche oder benutzerdefinierten Metadatenfeldern.

    1. Wählen Sie die Option Benutzerdefiniert aus.
    2. Suchparameter können auf Einschließen oder Ausschließen von Videos basierend auf Tags, Suche eingestellt werden oder ein benutzerdefiniertes Feld.
      Hinweis: Wenn Sie die Option Benutzerdefiniertes Feld verwenden, sollten Sie den internen Namen des benutzerdefinierten Feldes eingeben, NICHT die Anzeige benennen.
    3. Konfigurieren Sie die entsprechenden Suchparameter, um die gewünschten Videos zurückzugeben. Im folgenden Beispiel werden Videos zurückgegeben, die mit Verkaufsschulungen getaggt wurden.
    4. (Optional) Klicken Sie auf das Symbol „Hinzufügen“ () im entsprechenden Abschnitt, um zusätzliche Parameter einzuschließen oder auszuschließen.
    5. Legen Sie die Sortierreihenfolge für Videos fest
      • Aktuellste - Videos, die basierend auf dem Zeitpunkt der Aktivierung bestellt wurden, die neuesten wurden zuerst aufgeführt
      • Älteste zuerst - Videos, die nach ihrer Aktivierung bestellt wurden, die ältesten zuerst aufgeführt
      • Am meisten angesehen - Die meisten Ansichten aller Zeiten
      • A-Z - In aufsteigender Reihenfolge sortiert
      • Z-A - In absteigender Reihenfolge sortiert
    6. Klicken Sie auf Speichern, um Ihre Änderungen zu speichern.
