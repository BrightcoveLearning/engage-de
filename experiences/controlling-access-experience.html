Kontrollieren des Zugriffs auf eine Erfahrung

In diesem Thema erfahren Sie, wie Sie den Zugriff auf eine Erfahrung mithilfe von Zugriffssteuerungsprofilen einschränken.

Zugriffssteuerungsprofile werden zur Steuerung des Zugriffs auf Erlebnisse verwendet und als Teil der Projekteinstellungen erstellt. Sobald Profile erstellt wurden, können sie Erfahrungen zugewiesen werden, um den Zugriff zu steuern. Informationen zum Erstellen von Zugriffssteuerungsprofilen finden Sie unter Erstellen von Zugriffssteuerungsprofilen.

Gehen Sie folgendermaßen vor, um einer Erfahrung ein Zugriffssteuerungsprofil zuzuweisen:

  1. Bearbeiten Sie das Erlebnis.
  2. Klicken STANDORTKONFIGURATION> Zugriffskontrolle in der linken Navigation.
  3. Wählen Sie ein Zugriffssteuerungsprofil für die Erfahrung.
  4. Klicken Sie auf Speichern.

Stellen Sie sicher, dass Sie die Erfahrung erneut veröffentlichen, nachdem Sie Änderungen an den Einstellungen für die Zugriffssteuerung vorgenommen haben.
