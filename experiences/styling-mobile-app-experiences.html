Styling Mobile App-Erlebnisse

In diesem Thema erfahren Sie, wie Sie eine Mobile App Experience gestalten.

Mit dem Experience Editor können Sie den Stil einer Mobile App Experience anpassen, indem Sie das Thema und die Farben ändern. Um den Stil zu bearbeiten, bearbeiten Sie die Erfahrung und klicken Sie dann auf STIL in der linken Navigation.

Auswahl eines Themas

Auswählen von a Thema Wendet einen vordefinierten Satz von Farben auf das Erlebnis an. Um es zu ändern Thema::

  1. Klicken Sie im linken Navigationsbereich auf STYLE.
  2. Wählen Sie ein Thema aus der Dropdown-Liste aus. Wenn Sie ein Thema auswählen, wird die Vorschau aktualisiert.
  3. Um zum Experience Editor zurückzukehren, klicken Sie auf < Zurück zu Experience.

Farben ändern

Die Farbeinstellungen können verwendet werden, um die Farben, die von einem Thema bereitgestellt werden, fein abzustimmen. Zum Beispiel können Farbeinstellungen verwendet werden, um die Hintergrund- und Schriftfarben fein abzustimmen. Folgende Farben können geändert werden:

  • Akzent - Steuert die Hintergrundfarbe beim Laden der App, die Farbe der Leiste oben in der App und die Schriftfarbe für die Sammlungsnamen auf der Startseite
  • Primärer Text - Steuert die Schriftfarbe des Alle ansehen Schaltflächen- und Videotitel, die unter den Videoposterbildern auf der Startseite angezeigt werden
  • Sekundärtext - Steuert die Farbe der Titel, die auf den Seiten Sammlungen, Videodetails und Profile angezeigt werden
  • Symbole - Steuert die Farbe der Navigations-, Such- und Benutzersymbole oben in der App
  • Hintergrund - Steuert die Hintergrundfarbe der App

So passen Sie die Farben an:

  1. Klicken Sie im linken Navigationsbereich auf das STIL-Symbol. Das Farben Im Abschnitt werden alle verfügbaren Objekte und ihre Farben angezeigt.
  2. Um eine neue Farbe auszuwählen, klicken Sie auf einen Farbkreis.
  3. Wählen Sie mit der Farbauswahl eine Farbe aus, indem Sie auf die Farbe im Farbquadrat klicken oder eine hexadezimale Farbzahl unter dem Farbquadrat eingeben. Beachten Sie, dass Sie den Farbbereich ändern können, indem Sie in das Farbspektrum unter dem Farbquadrat klicken. Klicken Sie auf X, um die Farbauswahl zu schließen und die Änderung zu speichern.

    Um eine Farbauswahl zurückzusetzen, klicken Sie auf den Pfeil nach links neben dem Farbkreis.

  4. Um zum Experience Editor zurückzukehren, klicken Sie auf < Zurück zu Experience.
