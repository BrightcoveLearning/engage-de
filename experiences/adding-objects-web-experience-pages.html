Hinzufügen von Bildern, Text und Links zu Web-Experience-Seiten

In diesem Thema erfahren Sie, wie Sie Bilder, Texte und Links zu Web-Experience-Seiten hinzufügen.

Objekte wie Bilder, Text und Links können zu Web-Experience-Seiten hinzugefügt werden, sodass Sie Ihr Videoerlebnis anpassen und markieren können.

Objekte zu Seiten hinzufügen

Mit dem Experience Editor können Sie Objekte wie Bilder, Text und Links zu einer Seite hinzufügen. Um die Seiten zu bearbeiten, bearbeiten Sie das Erlebnis und klicken Sie dann in der linken Navigation auf SEITEN. Die Seitenvorlage zeigt Platzhalter auf der Seite und ein Bleistiftsymbol () zeigt Objekte an, die bearbeitet werden können. Diese Platzhalter können verwendet werden, um Ihrer Seite Bilder, Text und Links hinzuzufügen.

Text zu einer Seite hinzufügen

Text kann auf verschiedene Arten zu einer Seite hinzugefügt werden:

  • Durch Klicken auf den Link +Inhalt hinzufügen oben auf der Seite
  • Durch Klicken auf einen der Textplatzhalter auf der Seite

Fügen Sie Text hinzu, indem Sie auf den Link +Inhalt hinzufügen klicken

So fügen Sie der Seite Text hinzu, indem Sie auf den Link +Inhalt hinzufügen klicken:

  1. Klicken Sie auf den Link +Inhalt hinzufügen.
  2. Wählen Sie Text aus und klicken Sie dann auf Speichern.
  3. Klicken Sie auf den Platzhalter, der der Seite hinzugefügt wird, und geben Sie Ihren Text ein. Klicken Sie auf Speichern wenn Sie fertig sind.

Fügen Sie Text hinzu, indem Sie auf einen Textplatzhalter klicken

Seiten enthalten zahlreiche Textobjekte, in denen der Seite benutzerdefinierter Text hinzugefügt werden kann. Ein Bleistiftsymbol zeigt an, dass das Element bearbeitet werden kann.

So fügen Sie Ihren Text hinzu:

  1. Klicken Sie auf eines der Platzhalterobjekte.
  2. Geben Sie Ihren Text ein und klicken Sie auf Speichern.

Hinzufügen eines Bildes zum Seitenkopf

Die Größe der Bilder wird basierend auf der Vorlage geändert (nie hochskaliert). Der Bild-Uploader zeigt möglicherweise empfohlene Bildmaße an, je nachdem, wo das Bild verwendet wird. JPEG-, PNG- und GIF-Bilder werden unterstützt. So fügen Sie dem Seitenkopf ein Bild hinzu:

  1. Klicken Sie auf den Link +Inhalt hinzufügen.
  2. Wählen Sie Hochladen aus und klicken Sie dann auf Speichern.
  3. Klicken Sie auf den Link +Foto hinzufügen. Der Image-Browser-Dialog wird geöffnet.
  4. Um ein neues Bild hochzuladen, klicken Sie auf die Schaltfläche Datei auswählen und suchen Sie dann ein Bild aus dem Dateisystem aus. Sie können auch Drag & Drop und Bild machen. Klicken Sie auf Speichern wenn Sie fertig sind.
    Hinweis: Brightcove wird die Größe des Bildes auf 140 x 50 Pixel ändern.

Hinzufügen von Navigationslinks zur Seitenfußzeile

Navigationslinks können zur Seitenfußzeile hinzugefügt werden. Dies ist nützlich, wenn Sie einen Titel und dann eine Reihe von Links unten haben möchten. Zum Beispiel:

Das Navigationsobjekt der Seitenfußzeile besteht aus zwei Teilen, einem PLATZHALTER für den Titel und +Add more für die Links.

So fügen Sie Navigationslinks zur Seitenfußzeile hinzu:

  1. Klicken Sie auf PLATZHALTER und geben Sie einen Wert für die Kopfzeile ein.
  2. Klicken Sie auf +Weitere hinzufügen und geben Sie Text und Link ein. Sie können auch die Reihenfolge der Links angeben, indem Sie den Wert Listenreihenfolge festlegen, wenn der Link erstellt wird.
  3. Klicken Sie auf Speichern.

Weitere Navigationslinks können durch Klicken auf +Weitere hinzufügen eingegeben werden.

Löschen von Objekten von einer Seite

Text, Bilder und Links können auf verschiedene Arten von der Seite entfernt werden.

So entfernen Sie Text oder Bilder aus dem Seitenkopf

  1. Klicken Sie auf das Pfeilsymbol für die Verfolgung in der Kopfzeile, die mit dem Text oder Bild verknüpft ist.

So entfernen Sie Navigationslinks aus der Seitenfußzeile

  1. Klicken Sie auf den Link in der Seitenfußzeile.
  2. Klicken Sie auf Entfernen.

So entfernen Sie den Titel aus der Seitenfußzeile

  1. Klicken Sie in der Seitenfußzeile auf den Titel.
  2. Löschen Sie den Titeltext und klicken Sie auf Speichern.
