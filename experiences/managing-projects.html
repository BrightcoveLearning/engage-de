Projekte verwalten

In diesem Thema erfahren Sie, wie Sie Projekte verwalten.

Klicken Sie zum Erstellen und Verwalten von Projekten auf Projekte im Navigationsmenü.

Auf der Seite Projekte können Sie:

  • Erstellen und verwalten Sie Erfahrungen
  • Einstellungen konfigurieren

Erfahrungen schaffen und verwalten

Beim Öffnen der Projekte Seite, die Erfahrungen Die Registerkarte wird standardmäßig angezeigt. Wenn keine Erfahrungen erstellt wurden, werden Sie aufgefordert, eine neue Erfahrung zu erstellen. Eine Übersicht über die Arten von Erfahrungen, die erstellt werden können, finden Sie unter Übersicht über Erfahrungsvorlagen.

Weitere Informationen zum Erstellen und Verwalten von Erfahrungen finden Sie unter Erfahrungen erstellen und verwalten.

Einstellungen konfigurieren

Das die Einstellungen Auf der Registerkarte können Einstellungen für Erlebnisse konfiguriert werden. Weitere Informationen zu den Einstellungen finden Sie unter Konfigurieren der Erfahrungseinstellungen.
