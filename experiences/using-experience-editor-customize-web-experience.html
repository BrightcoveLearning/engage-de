Verwenden des Experience Editors zum Anpassen einer Web Experience

In diesem Thema erfahren Sie, wie Sie mit dem Experience Editor das Erscheinungsbild einer Web Experience anpassen.

Der Experience Editor wird verwendet, um das Erscheinungsbild von Erfahrungen anzupassen, die mit Brightcove Engage erstellt wurden™. Klicken Sie auf den Erfahrungstitel, um den Experience Editor zu öffnen. Mit Web Experiences bietet der Experience Editor die Möglichkeit:

  • Objekte zu Seiten hinzufügen
  • Fügen Sie einer Site Videos hinzu
  • Passen Sie den Site-Stil an
  • Kontrollieren Sie den Zugriff auf die Site
  • Konfigurieren Sie die Site-Details

Auf diese Optionen wird zugegriffen, indem Sie auf die Menüs in der linken Navigation klicken.

Hinzufügen von Objekten zu einer Seite

Objekte wie Text, Bilder und URLs können zu Seiten hinzugefügt werden, sodass Sie Ihre Web-Erfahrung anpassen und mit einem Branding versehen können. Informationen zum Hinzufügen von Objekten zu Seiten finden Sie unter Hinzufügen von Bildern, Text und Links zu Web Experience-Seiten.

Hinzufügen von Videos zu einer Web-Erfahrung

Um Videos zu einem Erlebnis hinzuzufügen, klicke auf VIDEOS in der linken Navigation. Weitere Informationen zum Hinzufügen von Videos zu einer Web Experience finden Sie unter Hinzufügen von Videos zu einer Erfahrung.

Anpassen des Stils

Klicken Sie auf, um die in einer Web Experience verwendeten Farben und Schriftarten anzupassen oder ein anderes Thema auszuwählen STIL in der linken Navigation. Einzelheiten zum Anpassen des Stils finden Sie unter Anpassen des Stils einer Web-Erfahrung.

Kontrollieren des Zugriffs auf die Site

Die Einstellungen für die Zugriffssteuerung werden verwendet, um den Zugriff auf die Site einzuschränken. Um die Einstellungen für die Zugriffssteuerung zu konfigurieren, klicken Sie auf ZUGANGSKONTROLLE in der linken Navigation.

Einzelheiten zu den Einstellungen für die Zugriffskontrolle finden Sie unter Kontrollieren des Zugriffs auf eine Erfahrung.

Konfigurieren der Erfahrungsdetails

Die Übersichtseinstellungen bieten die Möglichkeit, den Namen, die Beschreibung und die Sprache für die Erfahrung zu bearbeiten. Um die Erlebnisdetails zu konfigurieren, klicken Sie auf ÜBERBLICK in der linken Navigation.

Einzelheiten zum Konfigurieren der Site-Details finden Sie unter Erfahrungsdetails bearbeiten.
