Erfahrungsdetails bearbeiten

In diesem Thema erfahren Sie, wie Sie die Details für eine Erfahrung bearbeiten.

Der Titel, die Beschreibung und die Sprache der Erfahrung können als Teil der Übersichtseinstellungen konfiguriert werden. Um die Übersichtseinstellungen zu konfigurieren, bearbeiten Sie die Erfahrung und klicken Sie dann auf ÜBERBLICK in der linken Navigation.

Die Übersichtsseite wird angezeigt:

  • Erfahrungstyp - Zeigt die Art der Erfahrung an
  • Erleben Sie Titel und Beschreibung - Wenn Sie zum ersten Mal eine neue Erfahrung erstellen, werden Sie aufgefordert, die einzugeben Titel erleben und Erfahrungsbeschreibung ;; Diese Werte können hier bearbeitet werden
  • Sprache erleben - Setzt den spezifischen Text der Erfahrung auf diese Sprache (nur Web-Erfahrungen)
  • Datenschutz-Bestimmungen - Geben Sie die URL zu Ihrer Datenschutzrichtlinie ein (nur Mobile App).
  • Geschäftsbedingungen - Geben Sie die URL zu Ihren Allgemeinen Geschäftsbedingungen ein (nur Mobile App)
  • Hilfe Kontakt E-Mail - Geben Sie die E-Mail-Adresse ein, die verwendet werden soll, wenn Benutzer Probleme mit der Verwendung der App haben (nur mobile App).

Wenn Sie fertig sind, klicken Sie auf speichern um Ihre Änderungen zu speichern. Beachten Sie, dass auf der Profilseite der mobilen App Links für die Datenschutzrichtlinie, die Allgemeinen Geschäftsbedingungen und die Hilfekontakt-E-Mail angezeigt werden.

Hinweis: Nachdem Sie Änderungen an einem Erlebnis vorgenommen haben, stellen Sie sicher, dass Sie das Erlebnis erneut veröffentlichen.
