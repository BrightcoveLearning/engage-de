Verwenden des Experience Editors zum Anpassen einer Mobile App Experience

In diesem Thema erfahren Sie, wie Sie mit dem Experience Editor das Erscheinungsbild einer Mobile App Experience anpassen.

Der Experience Editor wird verwendet, um das Erscheinungsbild von Erfahrungen anzupassen, die mit Brightcove Engage erstellt wurden™. Klicken Sie auf den Erfahrungstitel, um den Experience Editor zu öffnen. Mit Mobile App Experiences bietet der Experience Editor folgende Möglichkeiten:

  • Fügen Sie der App ein Logo hinzu
  • Fügen Sie einer App Videos hinzu
  • Passen Sie den App-Stil an
  • Konfigurieren Sie die Zugriffssteuerung
  • Konfigurieren Sie die App-Details

Auf diese Optionen wird zugegriffen, indem Sie auf die Menüs in der linken Navigation klicken.

Hinzufügen eines Logos zur App

Am oberen Rand einer Mobile App Experience kann ein Logo hinzugefügt werden, mit dem Sie die Erfahrung brandmarken können. Informationen zum Hinzufügen eines Logos finden Sie unter Hinzufügen eines Logos zu einer mobilen App-Erfahrung.

Hinzufügen von Videos zu einer App

Um Videos zu einem Erlebnis hinzuzufügen, klicke auf VIDEOS in der linken Navigation. Weitere Informationen zum Hinzufügen von Videos zu einer Erfahrung finden Sie unter Hinzufügen von Videos zu einer Erfahrung.

Anpassen des App-Stils

Klicken Sie auf, um die Farben und das Thema anzupassen, die in einer Mobile App Experience verwendet werden STIL in der linken Navigation. Einzelheiten zum Anpassen des Stils finden Sie unter Styling Mobile App-Erlebnisse.

Zugriffskontrolle konfigurieren

Die Einstellungen für die Zugriffssteuerung bieten die Möglichkeit, den Zugriff auf die App einzuschränken. App-Benutzer müssen sich beim Öffnen der App authentifizieren. Um die Einstellungen für die Zugriffssteuerung zu konfigurieren, klicken Sie auf ZUGANGSKONTROLLE in der linken Navigation.

Einzelheiten zu den Einstellungen für die Zugriffskontrolle finden Sie unter Kontrollieren des Zugriffs auf eine Erfahrung.

Konfigurieren der App-Details

Die Übersichtseinstellungen bieten die Möglichkeit, den Namen, die Beschreibung und die Sprache für eine Erfahrung zu bearbeiten. Um die Erlebnisdetails zu konfigurieren, klicken Sie auf ÜBERBLICK in der linken Navigation.

Einzelheiten zum Konfigurieren der Erfahrungsdetails finden Sie unter Erfahrungsdetails bearbeiten.
