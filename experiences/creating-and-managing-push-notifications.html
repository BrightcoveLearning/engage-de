Erstellen und Verwalten von Push-Benachrichtigungen

In diesem Thema erfahren Sie, wie Sie Push-Benachrichtigungen erstellen und verwalten.

Eine Push-Benachrichtigung ist eine Nachricht, die auf einem mobilen Gerät angezeigt wird. Push-Benachrichtigungen sind eine bequeme Möglichkeit, Benutzer über neue Inhalte oder bevorstehende Live-Ereignisse zu informieren, die in Brightcove Engage verfügbar sind/werden.™ App. Wenn ein mobiler Benutzer auf eine Push-Benachrichtigung klickt, wird er zu den neuen Inhalten in der Brightcove Engage-App weitergeleitet.

Hinweis: Push-Benachrichtigungen können nur als Teil einer Mobile App-Erfahrung erstellt werden.

Benachrichtigungen verwalten

Um die Benachrichtigung für eine Mobile App-Erfahrung anzuzeigen, bearbeiten Sie die Mobile App-Erfahrung und klicken Sie dann auf MITTEILUNGEN in der linken Navigation. Eine Liste der Benachrichtigungen, die gesendet wurden oder geplant sind, wird angezeigt.

Der Hauptteil der Seite zeigt die folgenden Informationen an:

  • Benachrichtigungstitel - Titel der Meldung
  • Verknüpfter Inhalt - Der Inhalt, der angezeigt wird, wenn auf die Benachrichtigung geklickt wird (App-Startseite, Videosammlung, Video).
  • Benachrichtigungsdatum - Datum und Uhrzeit des Versendens der Benachrichtigung oder des geplanten Versands
  • Status
    • Geplant - Die Benachrichtigung soll gesendet werden
    • Geschickt - Die Benachrichtigung wurde gesendet

Durch Klicken auf eine Benachrichtigung werden die Benachrichtigungsdetails im Seitenbereich geöffnet.

Benachrichtigungen erstellen

Gehen Sie folgendermaßen vor, um eine neue Push-Benachrichtigung zu erstellen:

  1. Bearbeiten Sie die Mobile App-Erfahrung.
  2. Klicken MITTEILUNGEN in der linken Navigation. Eine Liste der Benachrichtigungen, die gesendet wurden oder geplant sind, wird angezeigt.
  3. Klicken Benachrichtigung erstellen.
  4. Geben Sie ein Benachrichtigungstitel und Beschreibung. Diese werden auf dem Gerät angezeigt, daher wird eine genaue Beschreibung empfohlen.
  5. Das Klicken Sie auf Senden an: Abschnitt bestimmt, was passiert, wenn die Benachrichtigung auf dem Gerät angeklickt wird:
    • Anwendung Home - Der Benutzer wird zur Startseite der App weitergeleitet
    • Sammlung - Der Benutzer wird zur ausgewählten Sammlung geleitet
    • Videoinhalt - Der Benutzer wird zum ausgewählten Video geleitet

Das Vorschau Der Bereich zeigt eine Vorschau an, wie die Benachrichtigung beim Senden aussehen wird.

Klicken Senden um die Benachrichtigung sofort zu senden oder Zeitplan Wählen Sie ein Datum und eine Uhrzeit aus und klicken Sie dann auf Bestätigen.

Benachrichtigungen aktivieren

Um Benachrichtigungen zu aktivieren, werden die Betrachter aufgefordert, Benachrichtigungen zuzulassen, wenn sie die Brightcove Engage-App installieren oder öffnen.
