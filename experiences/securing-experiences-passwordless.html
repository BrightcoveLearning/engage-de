Sichern von Erfahrungen mit der kennwortlosen Authentifizierung von Auth0

In diesem Thema erfahren Sie, wie Sie Erfahrungen mit der kennwortlosen Auth0-Authentifizierung sichern.

Erfahrungen können mit Auth0 mit passwortloser Authentifizierung gesichert werden. Bei Verwendung dieser Sicherheitsmethode werden Benutzer aufgefordert, eine Telefonnummer einzugeben. Auth0 sendet einen Code zur einmaligen Verwendung mit Twilio. Der Benutzer gibt dann den Code in Ihre Anwendung ein. Wenn die an den Code angehängte Telefonnummer mit einem vorhandenen Benutzer übereinstimmt, authentifiziert Auth0 den Benutzer.

Hinweis: Die Schritte und Bilder in diesem Thema verwenden Testversionen von Auth0 und Twilio.

Bevor du anfängst

Führen Sie die folgenden Aufgaben aus, bevor Sie Auth0 Passwordless konfigurieren:

  • Konfigurieren Sie Ihre Mobile App-Erfahrung mit Auth0 wie in beschrieben Sichern von Erfahrungen mit Auth0
  • Melden Sie sich für eine Twilio Testkonto und verifizieren Sie Ihr Konto

Eine Twilio-Telefonnummer erhalten

Sie benötigen eine Twilio-Telefonnummer, die bei der Konfiguration von Passwordless als Absendernummer verwendet wird.

  1. Melden Sie sich bei Ihrem Twilio-Konto an.
  2. Auf der Instrumententafel Seite, klicken Sie Holen Sie sich eine Testnummer.
  3. Klicken Wählen Sie diese Nummer.
  4. Klicken Getan um die Nummer zu bestätigen.

Konfigurieren einer kennwortlosen Verbindung in Auth0

Beginnen Sie mit der Konfiguration einer kennwortlosen Verbindung in Auth0.

  1. Melden Sie sich bei Ihrem Auth0-Konto an.
  2. Klicken Sie in der linken Navigation auf Verbindungen> Passwortlos.
  3. Aktivieren Sie die SMS Verbindung.

Eingabe Ihrer Twilio SID

  1. Zurück zur Seite Twilio.
  2. Suchen Sie auf der Dashboard-Seite die ACCOUNT SID Feld.
  3. Kopieren Sie den Wert in die Zwischenablage.
  4. Zurück zur Seite Auth0.
  5. Fügen Sie den Wert in das Feld ein Twilio SID Feld.

Eingabe Ihres Twilio Auth Token

  1. Zurück zur Seite Twilio.
  2. Suchen Sie auf der Dashboard-Seite die AUTH TOKEN Feld.
  3. Drücke den Show Link, um den Token zu enthüllen.
  4. Kopieren Sie den Wert in die Zwischenablage.
  5. Zurück zur Seite Auth0.
  6. Fügen Sie den Wert in das Feld ein Twilio Auth Token Feld.

Eingabe Ihrer Twilio From-Telefonnummer

  1. Zurück zur Seite Twilio.
  2. Suchen Sie auf der Dashboard-Seite die VERSUCHSNUMMER Sie früher erstellt.
  3. Kopieren Sie die Nummer in die Zwischenablage.
  4. Zurück zur Seite Auth0.
  5. Fügen Sie den Wert in das Feld ein Von Feld.
  6. (Optional) Customize the Message text.

Aktivieren Sie Ihre Anwendung

  1. Drücke den Anwendungen Link oben im Dialog.
  2. Aktivieren Sie die von Ihnen erstellte Auth0-Anwendung, um Ihre Brightcove Engage-Anwendung zu sichern.
  3. Klicken speichern und schließen Sie den Dialog.

Konfigurieren Sie Ihre Anwendung für die Verwendung eines kennwortlosen Logins

  1. Klicken Sie in der linken Navigation auf Universeller Login.
  2. Drücke den Anmeldung Verknüpfung.
  3. Aktivieren Sie die Anmeldeseite anpassen Möglichkeit.
  4. Wählen Sperre (passwortlos) von dem STANDARDVORLAGEN Dropdown-Liste.
  5. Klicken ÄNDERUNGEN SPEICHERN.

Vorschau des Erlebnisses

Sobald die oben genannten Schritte ausgeführt wurden, können Sie eine Vorschau Ihrer Erfahrung anzeigen, um zu bestätigen, dass der Betrachter zur Eingabe einer Telefonnummer aufgefordert wird.

  1. Melden Sie sich bei Brightcove Engage an.
  2. Klicken Sie auf den Erfahrungstitel, der mit Auth0 gesichert wurde.
  3. Vorschau der Erfahrung. Es sollte ein Anmeldebildschirm angezeigt werden, in dem Sie zur Eingabe einer Telefonnummer aufgefordert werden.
  4. Schließen Sie die Vorschau.

 

Twilio gültige Telefonnummern hinzufügen

Fügen Sie Twilio zu Testzwecken eine oder mehrere Telefonnummern hinzu.

  1. Zurück zur Seite Twilio.
  2. Klicken Sie in der linken Navigation auf Alle Produkte und Dienstleistungen Symbol.
  3. Klicken Sie in der linken Navigation auf Telefonnummern.
  4. Klicken Sie in der linken Navigation auf Verifizierte angerufene IDs.
  5. Geben Sie Zahlen ein, die Sie zum Testen verwenden möchten. Ihr Testkonto sollte die Telefonnummer haben, die beim Erstellen Ihres Testkontos verwendet wurde.

Überprüfung der Konfiguration

Der letzte Schritt besteht darin, zu testen, ob sich Zuschauer mit ihrer Handynummer anmelden können.

  1. Öffnen Sie die Brightcove Engage-Anwendung auf Ihrem Gerät. Wenn Sie dazu aufgefordert werden, geben Sie die ein Zugangscode für Ihre Erfahrung.
  2. Geben Sie Ihre Telefonnummer ein und klicken Sie auf EINREICHEN.
  3. Bestätigen Sie, dass Sie eine SMS mit einem Bestätigungscode erhalten.
  4. Geben Sie den Bestätigungscode ein und klicken Sie auf EINREICHEN.

Ihre mobile Brightcove Engage-App sollte angezeigt werden.
