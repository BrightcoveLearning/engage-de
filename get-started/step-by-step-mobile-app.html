Schritt für Schritt: Erstellen und Veröffentlichen einer mobilen App-Erfahrung

Dieses Thema enthält eine Reihe von Schritten, die Sie zum Erstellen und Veröffentlichen einer mobilen App-Erfahrung mit Brightcove Engage ausführen können™.

Nachdem Sie dieses Thema abgeschlossen haben, sollten Sie in der Lage sein:

  • Verwenden Sie Brightcove Engage, um eine neue Mobile App-Erfahrung zu erstellen
  • Fügen Sie Videos zu einem Erlebnis hinzu
  • Passen Sie ein Erlebnis an
  • Vorschau und Veröffentlichung eines Erlebnisses
  • Installieren Sie die Brightcove Engage-App auf einem Gerät und greifen Sie auf eine Erfahrung zu
Hinweis: Eine schrittweise Anleitung zum Erstellen einer Web-Erfahrung finden Sie unter Schritt für Schritt: Erstellen und Veröffentlichen eines Weberlebnisses.

Publikum

Jeder, der native mobile Apps erstellen muss, um Unternehmensvideoinhalte anzuzeigen.

Voraussetzungen

  • Zugriff auf Brightcove Engage (Melden Sie sich für eine Testversion an)

Bevor du anfängst

In dieser Schritt-für-Schritt-Anleitung werden die Videos verwendet, die im Brightcove Engage-Testkonto enthalten sind. Fühlen Sie sich frei, Ihre eigenen Videos hochzuladen und zu verwenden.

Installieren der Brightcove Engage-App

Die mobile Brightcove Engage-App ist erforderlich, um veröffentlichte Mobile App-Erfahrungen anzuzeigen. Befolgen Sie diese Schritte, um die App zu installieren.

  1. Öffne das Appstore (iOS) oder die Google Play Store (Android) auf Ihrem Gerät.
  2. Suche nach Brightcove Engagement App.
  3. Laden Sie die App herunter und installieren Sie sie auf Ihrem Gerät.

Die App wird später verwendet, um die von uns erstellte Erfahrung anzuzeigen.

Erstellen einer mobilen App-Erfahrung

Um eine mobile App-Erfahrung mit Brightcove Engage zu erstellen, führen Sie die folgenden Aufgaben aus:

  • Organisieren Sie Videos mit Wiedergabelisten
  • Erstellen Sie eine neue Erfahrung
  • Bearbeiten Sie die Site-Details
  • Fügen Sie der Hauptseite ein Logo hinzu
  • Erstellen Sie Sammlungen und fügen Sie den Sammlungen Videos hinzu
  • Vorschau des Erlebnisses
  • Passen Sie den Stil an
  • Veröffentlichen Sie das Erlebnis
  • Greifen Sie auf die Erfahrung auf einem Gerät zu

Organisieren von Videos mit Wiedergabelisten

Bevor Sie mit dem Erstellen von Erlebnissen beginnen, ist es hilfreich darüber nachzudenken, wie Sie Ihre Videos organisieren/gruppieren möchten. In der Regel möchten Sie verwandte Videos zusammenfassen. Beispielsweise möchten Sie möglicherweise die CEO-Videos, die HR-Videos zusammen, Marketingvideos usw. gruppieren. Wenn Zuschauer Ihre Mobile App Experience verwenden, sehen sie eine Liste dieser Gruppen und wählen dann die Videos aus, die sie ansehen möchten.

Wenn Sie eine Mobile App Experience erstellen, sollten Sie außerdem ein einzelnes Video als "empfohlenes Video" auswählen. Dieses Video wird oben in der App gut sichtbar angezeigt. Bei Verwendung von Brightcove Engage werden verwandte Videos mithilfe von Playlists gruppiert.

Hinweis: Wenn du deine eigenen Videos verwendest, erstelle mehrere Wiedergabelisten, um die Videos zu organisieren.

Gehen Sie folgendermaßen vor, um die Beispielvideos in Wiedergabelisten zu organisieren.

  1. Melden Sie sich bei Brightcove Engage an. Die Projekte Seite erscheint.
  2. Drücke den Videos Registerkarte in der Kopfzeile der Navigation. Eine Liste der Beispielvideos wird angezeigt.
  3. Klicken Sie in der linken Navigation auf + Playlist erstellen.
  4. Stellen Sie den Playlist-Typ auf Manuell ein.
  5. Setzen Sie den Namen der Wiedergabeliste auf Featured Video.
  6. Markiere Playlist in Sidebar anzeigen.
  7. Klicken Sie auf Speichern.
  8. Erstellen Sie mit den gleichen Einstellungen zwei weitere Wiedergabelisten Sales Videos und HR Videos. Du solltest drei Wiedergabelisten erstellen lassen.
  9. Klicken Sie in der linken Navigation auf Alle Videos.
  10. Klicken Sie auf das Kontrollkästchen neben dem 2019 All Hands Meeting Video und dann klick Mehr> Zur Wiedergabeliste hinzufügen.
  11. Wähle aus Empfohlenes Video Playlist und klicken Sie dann auf Hinzufügen.
  12. Wählen Sie die folgenden Videos aus und fügen Sie sie dem hinzu Verkaufsvideos Wiedergabeliste.
    • Treffen mit Vertriebsstrategie 2018
    • Vertriebshow-To - Entwicklung Ihrer Pipeline in neuen Märkten - Ep 02
    • Verkaufs-How-To - Neue Techniken für Upselling-Exkursionen - Ep 01
  13. Wählen Sie die folgenden Videos aus und fügen Sie sie dem hinzu HR-Videos Wiedergabeliste.
    • HR - Aufbau Ihres internen Kommunikationsplans - E02
    • HR - Onboarding neuer Mitarbeiter - E01

Alle Videos sind jetzt in Wiedergabelisten unterteilt. Wenn Sie in der linken Navigation auf einen Wiedergabelistennamen klicken, werden die Videos in dieser Wiedergabeliste angezeigt.

Weitere Informationen zu Playlists finden Sie im Erstellen und Verwalten von Wiedergabelisten Thema.

Eine neue Erfahrung schaffen

Führen Sie die folgenden Schritte aus, um eine neue Mobile App Experience zu erstellen.

  1. Klicken Sie im Navigationskopf auf das Projekte Tab.
  2. Bestätige das Erfahrung schaffen Seite erscheint. Wenn eine Liste mit Erlebnissen angezeigt wird, klicken Sie auf Neue Erfahrung um die anzuzeigen Erfahrung schaffen Seite.
  3. Suchen Sie die App Vorlage und klicken Wählen.
    Hinweis: Sobald Sie eine Vorlage ausgewählt haben, kann sie nicht geändert werden. Weitere Informationsvorlagen finden Sie im Übersicht über Erlebnisvorlagen Thema.
  4. Eingeben Beispiel für eine mobile App für die Titel erleben. Für das Pre-Populate-Erlebnis? Einstellung, wählen Sie Nein danke aus.
  5. Klicken Sie auf Erlebnis erstellen.

Der Experience Editor wird geöffnet. Der Erfahrungseditor wird zum Anpassen der Erfahrung verwendet und bietet die Möglichkeit, Videos hinzuzufügen, Stile auszuwählen und andere Einstellungen zu konfigurieren. Alle Anpassungseinstellungen werden über das linke Navigationsmenü aufgerufen.

Bearbeiten der Site-Details

Der Erfahrungstitel, die Beschreibung und die Spracheinstellungen können mit dem Experience Editor bearbeitet werden.

  1. Klicken ÜBERBLICK in der linken Navigation.
  2. Für die Erfahrungsbeschreibung , hinzufügen Beispiel für eine mobile App.
  3. Klicken Sie auf Speichern.

Hinzufügen eines Logos zur Hauptseite

Jede Erfahrung besteht aus einer Reihe von Seitenvorlagen. Die bereitgestellten Seitenvorlagen hängen von der Vorlage ab, die beim ersten Erstellen der Erfahrung ausgewählt wurde. Die Mobile App-Vorlage bietet Zuhause , Sammlungen , Videodetails , Suche und Profil Seitenvorlagen. Diese Seiten sind am linken Rand als Teil des PAGE-Menüs zu sehen.

Die Mobile App-Vorlage bietet die Möglichkeit, der Startseite ein Logo hinzuzufügen. Informationen zum Hinzufügen eines Logos finden Sie unter Hinzufügen eines Logos zu einer mobilen App-Erfahrung.

Führen Sie die folgenden Schritte aus, um der Startseite ein Logo hinzuzufügen.

  1. Klicken SEITEN> Heim in der linken Navigation.
  2. Drücke den Logo hinzufügen Link zur Erlebnisvorschau.
  3. Klicken Durchsuche Durchsuchen Sie dann ein Bild, das in der Kopfzeile verwendet werden soll, und wählen Sie es aus.
    Hinweis: Brightcove wird die Größe des Bildes auf 65 x 30 Pixel ändern.
    Hinweis: Brightcove empfiehlt, Bilder mit einer Größe von < 20 KB zu verwenden.
  4. Klicken Sie auf Speichern.
  5. Stellen Sie sicher, dass das Bild in der Erlebnisvorschau angezeigt wird.

Sammlungen erstellen und Videos zu den Sammlungen hinzufügen

Mit dem Experience Editor können Sie auswählen, welche Videos in der Erfahrung angezeigt werden sollen. Beim Erstellen von Erlebnissen werden Videos in Sammlungen unterteilt. Beim Erstellen von mobilen App-Erlebnissen wird das erste Video in der ersten Sammlung als vorgestelltes Video verwendet. Das vorgestellte Video wird oben in der Anwendung angezeigt.

Beim Erstellen von Sammlungen für mobile App-Erlebnisse empfiehlt Brightcove, eine Featured Video-Sammlung als erste Sammlung zu erstellen. Wir werden dann die zuvor erstellte Wiedergabeliste für vorgestellte Videos dieser Sammlung zuweisen.

Weitere Informationen zum Erstellen von Sammlungen und zum Hinzufügen von Videos zu einem Erlebnis finden Sie unter Videos zu einem Erlebnis hinzufügen.

Gehen Sie folgendermaßen vor, um dem Erlebnis Videos hinzuzufügen.

  1. Klicken VIDEOS in der linken Navigation.
  2. Nennen Sie die erste Sammlung Empfohlenes Video und dann drücken eingeben.
  3. Klicken Sie auf den Sammlungsnamen, um den Sammlungseditor zu öffnen.
  4. Wählen Sie Playlist für die Kriterien aus.
  5. Drücke den Empfohlenes Video Wiedergabeliste. Das Video in dieser Wiedergabeliste wird in der Videoliste angezeigt. Klicken Sie auf Speichern.
    Hinweis: Die Empfohlenes Video Playlist wurde im . erstellt Organisieren von Videos mit Playlists Sektion.
    Hinweis: Bei Verwendung einer Wiedergabelisten-Videozuweisung basiert die Reihenfolge der Videos auf den Einstellungen der Wiedergabeliste, die auf der Videoseite festgelegt wurden.
  6. Erstellen Sie eine weitere Sammlung mit dem Namen Der Umsatz Verwendung der Verkaufsvideos Wiedergabeliste als Kriterium.
  7. Erstellen Sie eine weitere Sammlung mit dem Namen HR Verwendung der HR-Videos Playlist als Kriterium.
  8. Bestätigen Sie, dass 3 Sammlungen erstellt wurden.
    Hinweis: Sie können die Reihenfolge der Sammlungen ändern, indem Sie auf die Sammlung klicken und diese an eine neue Position ziehen.

Vorschau des Erlebnisses

In der Vorschau einer Mobile App Experience können Sie sehen, wie sie auf Mobilgeräten angezeigt wird. Eine schnelle Vorschau erhalten Sie, indem Sie auf die Seitenvorlagen klicken, die unter angezeigt werden SEITEN Navigationselement. Beachten Sie, dass bei Verwendung dieser Vorschauoption die Videos nicht abgespielt werden, Sie jedoch eine Ansicht des Layouts erhalten können.

Die Erfahrung kann auch in einer neuen Browser-Registerkarte in der Vorschau angezeigt oder ein gemeinsam nutzbarer Link generiert werden, damit andere eine Vorschau anzeigen können. Weitere Informationen zur Vorschau und Veröffentlichung einer Mobile App Experience finden Sie unter Vorschau und Veröffentlichung einer Mobile App Experience.

Gehen Sie folgendermaßen vor, um eine Vorschau der Erfahrung anzuzeigen.

  1. Klicken Sie beim Bearbeiten der Site mit dem Experience Editor auf Vorschau und dann klicken Reaktionsschnelle Vorschau.
  2. Die Vorschau wird in einem neuen Browser-Tab geöffnet.
  3. Schließen Sie die Registerkarte „Site-Vorschau“.

Anpassen des Stils

Mit den Stileinstellungen können Sie die Farben anpassen, die für die Erfahrung verwendet werden. Weitere Informationen zum Ändern des Stils für Mobile App Experiences finden Sie unter Styling Mobile App-Erlebnisse.

Führen Sie die folgenden Schritte aus, um den Stil zu ändern.

  1. Klicken Sie im linken Navigationsbereich auf STYLE.
  2. EIN Thema ist eine Reihe von Farben, die auf eine Vorlage angewendet werden können. Wähle aus Licht Thema.
  3. Um eine Farbe zu ändern, klicken Sie auf den zugehörigen Farbkreis.
  4. Verwenden Sie den Schieberegler unter dem Farbquadrat, um einen Farbbereich auszuwählen.
  5. Klicken Sie in das Quadrat, um einen Farbton der gewünschten Farbe auszuwählen. Die Farbe kann auch durch Eingabe des entsprechenden Hex-Codes, RGB-Werts oder HSV-Werts (Farbton, Sättigung, Helligkeit) in das Textsteuerelement eingestellt werden.
  6. Drücke den <Zurück zur Erfahrung Link oben im Stilmenü.

Klicken Sie auf den Pfeil nach links neben dem Farbkreis, um die Farbe auf den ursprünglichen Wert zurückzusetzen.

Das Erlebnis veröffentlichen

Mobile App Experiences müssen veröffentlicht werden, bevor auf mobile Geräte zugegriffen werden kann. Informationen zur Vorschau und Veröffentlichung einer Mobile App Experience finden Sie unter Vorschau und Veröffentlichung einer Mobile App Experience.

Gehen Sie folgendermaßen vor, um das Erlebnis zu veröffentlichen.

  1. Klicken Sie im Experience Editor auf Veröffentlichen.
  2. Klicken Sie im Dialogfeld Diese mobile Anwendung veröffentlichen auf Jetzt veröffentlichen um die Veröffentlichung zu bestätigen.

Der Zugriffsschlüssel wird benötigt, um auf mobile Geräte zugreifen zu können.

Zugriff auf die Erfahrung auf einem Gerät

Sobald die Mobile App Experience veröffentlicht wurde, wird der Zugriffsschlüssel benötigt, um auf die Erfahrung auf einem mobilen Gerät zuzugreifen.

Führen Sie die folgenden Schritte aus, um auf einem mobilen Gerät auf die Erfahrung zuzugreifen.

  1. Wenn Sie nicht über den Zugriffsschlüssel verfügen, wird der Zugriffsschlüssel als Teil der Erfahrungseigenschaften auf dem angezeigt Projekte Startseite. Klicken Sie auf die Erfahrung, um sie auszuwählen. Der Zugriffsschlüssel wird im Eigenschaftenbereich angezeigt.
  2. Öffnen Sie die Brightcove Engage-App auf Ihrem Gerät. Geben Sie die Zugangsschlüssel und dann klicken Nächster.
