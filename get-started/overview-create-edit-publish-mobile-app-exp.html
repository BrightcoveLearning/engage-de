Überblick: Erstellen, Bearbeiten und Veröffentlichen von Erfahrungen mit mobilen Apps

In diesem Thema lernen Sie die grundlegenden Schritte zum Erstellen, Bearbeiten und Veröffentlichen einer Mobile App Experience kennen.

Mit Brightcove Engage™ können Sie ansprechende Videoerlebnisse erstellen, ohne technische Ressourcen zu benötigen. Das folgende Beispiel zeigt eine Mobile App-Erfahrung, die mit der Brightcove Engage Mobile App-Vorlage erstellt wurde.

Eine schrittweise Anleitung zum Erstellen einer Mobile App Experience finden Sie unter Schritt für Schritt: Erstellen und Veröffentlichen einer mobilen App-Erfahrung.

Die wichtigsten Schritte zum Erstellen und Veröffentlichen einer Mobile App Experience sind:

  • Wählen Sie eine Vorlage
  • Fügen Sie Video hinzu und passen Sie das Erlebnis an
  • Vorschau und Veröffentlichung des Erlebnisses

Auswahl einer Vorlage

Wenn Sie ein neues Erlebnis erstellen, wählen Sie zunächst eine Vorlage aus. Die Vorlage steuert das allgemeine Erscheinungsbild des Erlebnisses. Um mehr über Vorlagen zu erfahren und einige Beispiel-Sites anzuzeigen, lesen Sie die Übersicht über Erlebnisvorlagen Thema.

Nachdem Sie eine Vorlage ausgewählt haben, haben Sie die Möglichkeit, Vorbesetzen die Erfahrung mit Beispielinhalten. Klicken Sie auf Ja, füllen Sie meine Erfahrungen vor füllt die Erfahrung mit Videos aus Ihrem Brightcove-Konto vorab aus. Auf diese Weise können Sie leichter erkennen, wie das Erlebnis aussehen wird, wenn Sie mit dem Anpassungsprozess beginnen.

Video hinzufügen und das Erlebnis anpassen

Nachdem Sie eine Erfahrung erstellt haben, wird die Erfahrungseditor wird verwendet, um verschiedene Aspekte der Erfahrung anzupassen. Die folgenden Arten von Anpassungen können vorgenommen werden:

  • Seiten - Der Startseite kann ein Logo hinzugefügt werden, um das Erlebnis nach Bedarf zu kennzeichnen. Weitere Informationen zum Hinzufügen eines Logos finden Sie unter Hinzufügen eines Logos zu einer mobilen App-Erfahrung.
  • Videos - Sammlungen werden erstellt, um Videoinhalte auszuwählen und zu organisieren. Weitere Informationen zum Hinzufügen von Videoinhalten zur Erfahrung finden Sie unter Hinzufügen von Videos zu einer Erfahrung.
  • Stil - Das Erscheinungsbild der Erfahrung kann durch Auswahl eines neuen Themas oder durch Ändern der Site-Farben angepasst werden. Weitere Informationen zum Anpassen des Stils finden Sie unter Styling Mobile App-Erlebnisse.
  • Zugangskontrolle - Sie können den Zugriff auf die veröffentlichte App mit Single Sign-On einschränken. Weitere Informationen zur Zugriffskontrolle finden Sie unter Kontrollieren des Zugriffs auf eine Erfahrung.
  • Überblick - Sie können den Namen, die Beschreibung und die Sprache für die Erfahrung bearbeiten. Weitere Informationen zum Bearbeiten der Website-Übersicht finden Sie unter Experience Details bearbeiten.

Vorschau und Veröffentlichung des Erlebnisses

In der Vorschau einer mobilen App-Erfahrung können Sie sehen, wie die Erfahrung auf einem mobilen Gerät angezeigt wird. Erlebnisse können in einer neuen Browser-Registerkarte in der Vorschau angezeigt werden oder ein teilbarer Link kann generiert werden, damit andere eine Vorschau der Website anzeigen können. Durch das Veröffentlichen der Erfahrung wird ein Zugriffsschlüssel generiert, der vom Brightcove Engage mobile App. Weitere Informationen zum Anzeigen einer Vorschau und zum Veröffentlichen einer Erfahrung finden Sie unter Vorschau und Veröffentlichung einer Mobile App Experience.
