Brightcove Engage™ Best Practices

In diesem Thema erfahren Sie mehr über empfohlene Best Practices bei der Arbeit mit Brightcove Engage.

Mit Videos arbeiten

Brightcove Engage Testkonten haben folgende Einschränkungen:

  • Verwendung von manuellen Wiedergabelisten zum Organisieren von Videos
  • Verwenden dieser Wiedergabelisten für Sammlungen für beide Mobil-/Webportale, sodass Sie für beide nur eine Wiedergabeliste benötigen
  • So aktualisieren Sie die Projekte einfach, indem Sie beiden einfach eine Wiedergabeliste hinzufügen
  • Bildgrößen für jeden Projekttyp
  • Farbrauflinien für jeden
  • Der beste Weg, um die vorgestellte Playlist zu verwenden, um sowohl das erste Video auf dem Handy als auch das erste Set im Internet zu machen - optimale Nummer im Internet... Ich denke, wir haben 4 oder 8 gesagt? Es’liegt über dem Thread.
  • beginnend mit dem mobilen Portal
