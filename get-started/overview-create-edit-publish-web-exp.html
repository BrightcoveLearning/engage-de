Überblick: Erstellen, Bearbeiten und Veröffentlichen von Web-Erfahrungen

In diesem Thema lernen Sie die grundlegenden Schritte zum Erstellen, Bearbeiten und Veröffentlichen einer Web Experience kennen.

Mit Brightcove Engage™ können Sie ansprechende Videoerlebnisse erstellen, ohne technische Ressourcen zu benötigen. Web-Erlebnisse folgen reaktionsschnellen Designprinzipien, sodass sie auf Desktops, Tablets und Mobilgeräten gut aussehen.

Das folgende Beispiel zeigt eine Web-Erfahrung, die mit der Brightcove Engage-Webvorlage erstellt wurde.

Eine schrittweise Anleitung zum Erstellen einer Web Experience finden Sie unter Schritt für Schritt: Erstellen und Veröffentlichen einer Web-Erfahrung.

Die wichtigsten Schritte zum Erstellen und Veröffentlichen einer Web Experience sind:

  • Wählen Sie eine Vorlage
  • Fügen Sie Video hinzu und passen Sie das Erlebnis an
  • Vorschau und Veröffentlichung des Erlebnisses

Auswahl einer Vorlage

Wenn Sie ein neues Erlebnis erstellen, wählen Sie zunächst eine Vorlage aus. Die Vorlage steuert das allgemeine Erscheinungsbild des Erlebnisses. Brightcove Engage bietet eine Reihe von Vorlagen, mit denen Sie die Art von Videoerfahrung bereitstellen können, die für die Interaktion mit Ihren Zuschauern erforderlich ist. Um mehr über Vorlagen zu erfahren und einige Beispiel-Sites anzuzeigen, lesen Sie die Übersicht über Erlebnisvorlagen Thema.

Nachdem Sie eine Vorlage ausgewählt haben, haben Sie die Möglichkeit, Vorbesetzen die Erfahrung mit Beispielinhalten. Klicken Sie auf Ja, füllen Sie meine Erfahrungen vor wird die Erfahrung mit Videos aus Ihrem Konto vorab füllen. Auf diese Weise können Sie leichter erkennen, wie das Erlebnis aussehen wird, wenn Sie mit dem Anpassungsprozess beginnen.

Video hinzufügen und das Erlebnis anpassen

Nachdem Sie eine Erfahrung erstellt haben, wird die Erfahrungseditor wird verwendet, um die Erfahrung anzupassen. Die folgenden Arten von Anpassungen können vorgenommen werden:

  • Seiten - Bilder, Text und Links können zu Seitenvorlagen hinzugefügt werden, um das Erlebnis nach Bedarf zu kennzeichnen. Weitere Informationen zum Hinzufügen von Objekten zu Seiten finden Sie unter Anpassen des Inhalts von Web Experience-Seiten.
  • Videos - Sammlungen werden erstellt, um Videoinhalte auszuwählen und zu organisieren. Weitere Informationen zum Hinzufügen von Videoinhalten zur Website finden Sie unter Hinzufügen von Videos zu einer Erfahrung.
  • Stil - Das Erscheinungsbild der Erfahrung kann durch Auswahl eines neuen Themas oder durch Ändern der Site-Farben und -Schriftarten angepasst werden. Weitere Informationen zum Anpassen des Stils finden Sie unter Anpassen des Stils einer Web-Erfahrung.
  • Zugangskontrolle - Beschränken Sie den Zugriff auf die Site. Weitere Informationen zur Site-Konfiguration finden Sie unter Kontrollieren des Zugriffs auf eine Erfahrung.
  • Überblick - Bearbeiten Sie den Namen, die Beschreibung und die Sprache für die Erfahrung. Weitere Informationen zum Bearbeiten der Website-Übersicht finden Sie unter Experience Details bearbeiten.

Vorschau und Veröffentlichung des Erlebnisses

In der Vorschau einer Erfahrung können Sie sehen, wie die Erfahrung auf Mobil-, Tablet- und Desktopgeräten angezeigt wird. Erlebnisse können in einer neuen Browser-Registerkarte in der Vorschau angezeigt werden oder ein teilbarer Link kann generiert werden, damit andere eine Vorschau der Website anzeigen können. Durch das Veröffentlichen der Erfahrung wird eine statische URL für die Site generiert und für die Betrachter zugänglich gemacht. Weitere Informationen zum Anzeigen einer Vorschau und zum Veröffentlichen einer Web Experience finden Sie unter Vorschau und Veröffentlichung einer Web-Erfahrung.
