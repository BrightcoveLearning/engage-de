Erste Schritte mit Brightcove Engage™

In diesem Thema erfahren Sie, wie Sie in der Brightcove Engage-Anwendung navigieren.

Um auf Brightcove Engage zuzugreifen, navigieren Sie zu: https://studio.brightcove.com/products/brightcoveengage. Melden Sie sich mit Ihrer E-Mail-Adresse und Ihrem Passwort an. Wenn die Anwendung geöffnet wird, werden Sie zu der weitergeleitet Projekte Seite. Der Seitenkopf bietet folgende Möglichkeiten:

  • Anzeigen von Systembenachrichtigungen
  • Kontoeinstellungen verwalten
  • Wählen Sie ein Konto aus und sehen Sie sich das Benutzerprofil an
  • Greifen Sie auf Anwendungsseiten zu
Hinweis: Das Sitzungstimeout der Brightcove Engage-Anwendung beträgt 24 Stunden.

Anzeigen von Systembenachrichtigungen

In der Kopfzeile der Anwendung wird ein Glockensymbol () when there are system notifications available. Wenn Sie auf das Symbol klicken, wird die Liste der Systembenachrichtigungen angezeigt.

In der Kopfzeile wird ein Warnsymbol () when there are system issues. Durch Klicken auf das Symbol wird die Seite Brightcove-Systemstatus geöffnet.

Kontoeinstellungen verwalten

Das Administrationsmenü kann durch Klicken auf das Zahnradsymbol () in the header. Das Administrationsmenü bietet Zugriff auf:

  • Kontoinformationen - Informationen über das Girokonto anzeigen
  • Benutzerverwaltung - Verwalten Sie die Benutzer, die zur Verwendung der Anwendung berechtigt sind
  • Kontoeinstellungen - Verwalten der Berichtszeitzone
  • Videofelder - Anzeigen, Erstellen und Ändern von benutzerdefinierten Metadatenfeldern für das Konto
  • Teilen von Medien - Teilen Sie Videos mit anderen Konten
Hinweis: Sie müssen ein Kontoadministrator sein, um Zugriff auf das Verwaltungsmenü zu haben.

Auswahl eines Kontos

Wenn Sie Zugriff auf mehrere Konten haben, kann die Kontoauswahl verwendet werden, um das Konto auszuwählen, auf das Sie zugreifen möchten. Um das Konto zu wechseln, klicken Sie auf das Kontobezeichnung in der oberen rechten Ecke der Anwendung. Das Girokonto wird mit einem Punkt vor dem Kontonamen gekennzeichnet.

Drücke den Mein Profil Link, um Ihre Benutzerinformationen und Ihr Passwort zu aktualisieren. Weitere Informationen finden Sie unter Verwalten von Profilinformationen.

Zugriff auf Seiten

Der Anwendungsheader bietet Zugriff auf die verschiedenen Anwendungsseiten.

Die folgenden Anwendungsseiten sind verfügbar:

  • Projekte - Erstellen Sie Videoerlebnisse für Ihre Videos
  • Videos - Videos hochladen, Videos verwalten und Videoeigenschaften festlegen
  • Analytics - Anzeigen von Analysen zur Videoleistung

Lokalisierung

Die Anwendungssprache kann geändert werden, indem Sie die Sprache in Ihrem Profil ändern. Die folgenden Sprachen werden unterstützt:

  • Englisch
  • Spanisch (Spanien)
  • Japanisch
  • Französisch (Frankreich)
  • Chinesisch (Traditionell)
  • Koreanisch

Weitere Informationen finden Sie unter Verwalten von Profilinformationen.
