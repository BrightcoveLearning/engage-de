Schritt für Schritt: Erstellen und Veröffentlichen einer Web-Erfahrung

Dieses Thema enthält eine Reihe von Schritten, die Sie zum Erstellen und Veröffentlichen eines Weberlebnisses mit Brightcove Engage ausführen können™.

Nachdem Sie dieses Thema abgeschlossen haben, sollten Sie in der Lage sein:

  • Verwenden Sie Brightcove Engage, um eine neue Web-Erfahrung zu erstellen
  • Fügen Sie Videos zu einem Erlebnis hinzu
  • Passen Sie ein Erlebnis an
  • Vorschau und Veröffentlichung eines Erlebnisses
Hinweis: Eine schrittweise Anleitung zum Erstellen einer Mobile App Experience finden Sie unter Schritt für Schritt: Erstellen und Veröffentlichen einer mobilen App-Erfahrung.

Publikum

Jeder, der Webvideoportale erstellen muss, um Videoinhalte anzuzeigen.

Voraussetzungen

  • Ein Brightcove Engage-Konto (Melden Sie sich für eine Testversion an)

Bevor du anfängst

In dieser Schritt-für-Schritt-Anleitung werden die Beispielvideos verwendet, die im Brightcove Engage-Testkonto enthalten sind. Fühlen Sie sich frei, Ihre eigenen Videos hochzuladen und zu verwenden.

Erstellen einer Web-Erfahrung

Um eine Web-Erfahrung mit Brightcove Engage zu erstellen, führen Sie die folgenden Aufgaben aus:

  • Organisieren Sie Videos mit Wiedergabelisten
  • Erstellen Sie eine neue Web-Erfahrung
  • Bearbeiten Sie die Site-Details
  • Fügen Sie einer Seitenvorlage Bilder und Text hinzu
  • Erstellen Sie Sammlungen und fügen Sie den Sammlungen Videos hinzu
  • Vorschau des Erlebnisses
  • Ändern Sie den Site-Stil
  • Veröffentlichen Sie das Erlebnis

Organisieren von Videos mit Wiedergabelisten

Bevor Sie mit dem Erstellen von Erlebnissen beginnen, ist es hilfreich darüber nachzudenken, wie Sie Ihre Videos organisieren/gruppieren möchten. In der Regel möchten Sie verwandte Videos zusammenfassen. Beispielsweise möchten Sie möglicherweise die CEO-Videos, die HR-Videos zusammen, Marketingvideos usw. gruppieren. Wenn Zuschauer Ihre Web Experience verwenden, wird eine Liste dieser Gruppen angezeigt, eine Gruppe ausgewählt und anschließend die Videos ausgewählt, die sie ansehen möchten. Bei Verwendung von Brightcove Engage werden verwandte Videos mithilfe von Playlists gruppiert.

Hinweis: Wenn du deine eigenen Videos verwendest, erstelle mehrere Wiedergabelisten, um die Videos zu organisieren.

Gehen Sie folgendermaßen vor, um die Beispielvideos in Wiedergabelisten zu organisieren.

  1. Melden Sie sich bei Brightcove Engage an. Die Projekte Seite erscheint.
  2. Drücke den Videos Registerkarte in der Kopfzeile der Navigation. Eine Liste der Beispielvideos wird angezeigt.
  3. Klicken Sie in der linken Navigation auf + Playlist erstellen.
  4. Stellen Sie den Playlist-Typ auf Manuell ein.
  5. Setzen Sie den Namen der Wiedergabeliste auf Featured Video.
  6. Markiere Playlist in Sidebar anzeigen.
  7. Klicken Sie auf Speichern.
  8. Erstellen Sie mit den gleichen Einstellungen zwei weitere Wiedergabelisten Sales Videos und HR Videos. Du solltest drei Wiedergabelisten erstellen lassen.
  9. Klicken Sie in der linken Navigation auf Alle Videos.
  10. Klicken Sie auf das Kontrollkästchen neben dem 2019 All Hands Meeting Video und dann klick Mehr> Zur Wiedergabeliste hinzufügen.
  11. Wähle aus Empfohlenes Video Playlist und klicken Sie dann auf Hinzufügen.
  12. Wählen Sie die folgenden Videos aus und fügen Sie sie dem hinzu Verkaufsvideos Wiedergabeliste.
    • Treffen mit Vertriebsstrategie 2018
    • Vertriebshow-To - Entwicklung Ihrer Pipeline in neuen Märkten - Ep 02
    • Verkaufs-How-To - Neue Techniken für Upselling-Exkursionen - Ep 01
  13. Wählen Sie die folgenden Videos aus und fügen Sie sie dem hinzu HR-Videos Wiedergabeliste.
    • HR - Aufbau Ihres internen Kommunikationsplans - E02
    • HR - Onboarding neuer Mitarbeiter - E01

Alle Videos sind jetzt in Wiedergabelisten unterteilt. Wenn Sie in der linken Navigation auf einen Wiedergabelistennamen klicken, werden die Videos in dieser Wiedergabeliste angezeigt.

Weitere Informationen zu Playlists finden Sie im Erstellen und Verwalten von Wiedergabelisten Thema.

Erstellen einer neuen Web-Erfahrung

Nachdem die Videos in Wiedergabelisten organisiert sind, können wir eine neue Erfahrung erstellen. Führen Sie die folgenden Schritte aus, um eine neue Web-Erfahrung zu erstellen.

  1. Klicken Sie im Navigationskopf auf das Projekte Tab.
  2. Bestätige das Erfahrung schaffen Seite erscheint. Wenn eine Liste mit Erlebnissen angezeigt wird, klicken Sie auf Neue Erfahrung um die anzuzeigen Erfahrung schaffen Seite.
  3. Suchen Sie die Webvorlage und klicken Sie auf Wählen.
    Hinweis: Sobald Sie eine Vorlage ausgewählt haben, kann sie nicht geändert werden. Weitere Informationen zu Vorlagen finden Sie im Übersicht über Erlebnisvorlagen Thema.
  4. Eingeben Beispiel für eine Web Template-Erfahrung für die Titel erleben. Für das Pre-Populate-Erlebnis? Einstellung, wählen Sie Nein danke aus.
  5. Klicken Sie auf Erlebnis erstellen.

Der Experience Editor wird geöffnet. Der Erfahrungseditor wird zum Anpassen der Erfahrung verwendet und bietet die Möglichkeit, Seitenlayouts anzupassen, Videos hinzuzufügen, Stile auszuwählen und andere Einstellungen zu konfigurieren. Alle Anpassungseinstellungen werden über das linke Navigationsmenü aufgerufen.

Bearbeiten der Site-Details

Der Erfahrungstitel, die Beschreibung und die Spracheinstellungen können mit dem Experience Editor bearbeitet werden.

  1. Klicken ÜBERBLICK in der linken Navigation.
  2. Für die Erfahrungsbeschreibung , hinzufügen Beispiel für eine Web-Erfahrung.
  3. Klicken Sie auf Speichern.

Hinzufügen von Bildern und Text zu einer Seitenvorlage

Jede Webvorlage enthält eine Reihe von Seitenvorlagen, die angepasst werden können. Das Web Template bietet Zuhause , Kategorie und Video Detail Seitenvorlagen. Diese Seiten sind am linken Rand als Teil des PAGE-Menüs zu sehen.

Seitenvorlagen zeigen Platzhalter auf der Seite und ein Bleistiftsymbol () indicates that the object can be edited. Diese Platzhalter können verwendet werden, um Ihrer Seite Bilder, Text und Links hinzuzufügen.

Weitere Informationen zum Hinzufügen von Objekten zu Seitenvorlagen finden Sie unter Hinzufügen von Bildern, Text und Links zu Web Experience-Seiten.

Führen Sie die folgenden Schritte aus, um einer Seitenvorlage Objekte hinzuzufügen.

  1. Klicken SEITEN> Heim in der linken Navigation. Bleistiftsymbole () indicate areas on the page where images/text/links can be added. Statische Textobjekte auf der Seite können durch Klicken bearbeitet werden.
  2. Klicken Sie oben auf der Seite auf + Inhalt hinzufügen.
  3. Klick auf das Hochladen Option und klicken Sie dann auf speichern. Auswählen Hochladen fügt der Seite einen Bildplatzhalter hinzu.
  4. Klicke auf + Foto hinzufügen.
  5. Klicken Datei wählen und dann durchsuchen und auswählen und ein Bild für die Kopfzeile verwenden. Click Save.
    Hinweis: Brightcove wird die Größe des Bildes auf 140 x 50 Pixel ändern.
  6. Stellen Sie sicher, dass das Bild im Seitenkopf angezeigt wird.
  7. Klicken Sie unten auf der Seite auf PLATZHALTER auf der linken Seite der Fußzeile. Geben Sie Ihren Firmennamen ein und klicken Sie dann auf speichern.
  8. Klick auf das Kategorie und Video Detail Seiten am linken Rand. Beachten Sie, dass das Header-Bild und der Titeltext auch auf diesen Seiten angezeigt werden. Geh zurück zum Zuhause Seite.

Sammlungen erstellen und Videos zu den Sammlungen hinzufügen

Mit dem Experience Editor können Sie auswählen, welche Videos in der Erfahrung angezeigt werden sollen. Beim Erstellen von Erlebnissen werden Videos in Sammlungen unterteilt. In diesem Beispiel basieren Sammlungen auf den Wiedergabelisten, die vorhanden waren früher erstellt.

Weitere Informationen zum Erstellen von Sammlungen und zum Hinzufügen von Videos zu einem Erlebnis finden Sie unter Videos zu einem Erlebnis hinzufügen.

Gehen Sie folgendermaßen vor, um dem Erlebnis Videos hinzuzufügen.

  1. Klicken VIDEOS in der linken Navigation.
  2. Nennen Sie die erste Sammlung Der Umsatz und dann drücken eingeben.
  3. Klicken Sie auf den Sammlungsnamen, um den Sammlungseditor zu öffnen.
  4. Wählen Sie Playlist für die Kriterien aus.
  5. Drücke den Verkaufsvideos Wiedergabeliste. Die Videos in dieser Wiedergabeliste werden in der Videoliste angezeigt. Klicken Sie auf Speichern.
    Hinweis: Bei Verwendung einer Wiedergabelisten-Videozuweisung basiert die Reihenfolge der Videos auf den Einstellungen der Wiedergabeliste, die auf der Videoseite festgelegt wurden.
  6. Erstellen Sie eine weitere Sammlung mit dem Namen HR Verwendung der HR-Videos Playlist als Kriterium.
  7. Bestätigen Sie, dass 2 Sammlungen erstellt wurden.
    Hinweis: Sie können die Reihenfolge der Sammlungen ändern, indem Sie auf die Sammlung klicken und diese an eine neue Position ziehen.

Vorschau des Erlebnisses

In der Vorschau einer Erfahrung können Sie sehen, wie die Website auf Mobil-, Tablet- und Desktopgeräten angezeigt wird. Sie möchten eine Vorschau der Erfahrung anzeigen, während Sie die Schritte zum Anpassen ausführen. Die Erfahrung kann in einem neuen Browser-Tab in der Vorschau angezeigt oder ein gemeinsam nutzbarer Link generiert werden, damit andere das Portal in der Vorschau anzeigen können. Weitere Informationen zur Vorschau und Veröffentlichung einer Web-Erfahrung finden Sie unter Vorschau und Veröffentlichung einer Web-Erfahrung.

Gehen Sie folgendermaßen vor, um eine Vorschau der Erfahrung anzuzeigen.

  1. Klicken Sie während der Bearbeitung der Erfahrung auf Vorschau und dann klicken Reaktionsschnelle Vorschau.
  2. Die Vorschau wird in einem neuen Browser-Tab geöffnet.
  3. Schließen Sie die Registerkarte „Site-Vorschau“.

Ändern des Site-Stils

Mit den Stileinstellungen können Sie die Site-Farben und -Schriftarten anpassen, die für die Benutzererfahrung verwendet werden. Weitere Informationen zum Ändern des Site-Stils finden Sie unter Anpassen des Stils einer Web-Erfahrung.

Führen Sie die folgenden Schritte aus, um den Site-Stil zu ändern.

  1. Klicken Sie im linken Navigationsbereich auf STYLE.
  2. EIN Thema ist eine Reihe von Farben und Schriftarten, die auf eine Vorlage angewendet werden können. Wählen Sie a Thema. In diesem Beispiel wird das verwendet Blautöne Thema.
  3. Wählen Sie a Primär und Sekundär zu verwendende Schriftart. Das Seitentitel Schriftart steuert die in der Kopfzeile verwendete Titelschriftart. Wenn Sie diesem Schritt-für-Schritt-Dokument gefolgt sind, wurde der Kopfzeile zuvor ein Bild hinzugefügt.
  4. Drücke den <Zurück zur Erfahrung Link oben im Menü Stil.

Das Erlebnis veröffentlichen

Durch das Veröffentlichen eines Web-Erlebnisses wird es unter einer öffentlichen URL verfügbar. Weitere Informationen zum Anzeigen einer Vorschau und zum Veröffentlichen einer Portalerfahrung finden Sie unter Vorschau und Veröffentlichung einer Web-Erfahrung.

Gehen Sie folgendermaßen vor, um das Erlebnis zu veröffentlichen.

  1. Klicken Sie im Experience Editor auf Veröffentlichen.
  2. Klicken Sie im Dialogfeld "Site veröffentlichen" auf Jetzt veröffentlichen um die Veröffentlichung zu bestätigen.

Klicken Sie auf die Site-URL, um sie in einem anderen Browser-Tab zu öffnen.
