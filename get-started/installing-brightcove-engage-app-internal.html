Installieren von Brightcove Engage™ Bewerbung für interne Bewerbungen

In diesem Thema erfahren Sie, wie Sie die Brightcove Engage-Anwendung auf einem mobilen Gerät installieren.

Der in diesem Thema beschriebene Installationsprozess sollte verwendet werden, wenn Sie mithilfe von Brightcove Consulting eine eigene Marken-App erstellt haben. Apple und Google genehmigen möglicherweise keine internen Apps für ihre App Stores, für die Brightcove Engage von einem anderen Ort als einem App Store installiert werden muss.

Vor der Installation der Anwendung sollten Sie sicherstellen, dass die erforderlichen Installationsdateien für iOS und Android an einem adressierbaren Speicherort / einer adressierbaren URL gehostet werden.

Hinweis: Wenn Sie die Brightcove Engage-App aus einem der App Stores installieren möchten, lesen Sie dieses Dokument.

Durch das Veröffentlichen einer mobilen App-Erfahrung wird ein Zugriffsschlüssel generiert, der von der Brightcove Engage-Anwendung benötigt wird, um auf die Erfahrung auf mobilen Geräten zuzugreifen. Dieses Thema behandelt die Installation und grundlegende Navigation der Brightcove Engage Anwendung.

Installieren der nativen iOS-Anwendung

Die mobile Brightcove Engage App kann auf iOS-Geräten mit iOS 11 oder höher installiert werden. Befolgen Sie diese Schritte, um die App zu installieren.

  1. Öffnen Sie einen Browser und gehen Sie zu URL, in der sich die Installationsdatei befindet.
  2. Sie werden aufgefordert, die Installation zu bestätigen. Klicken Installieren. Die App wird auf Ihrem Gerät installiert.
  3. Da die App nicht aus dem Apple App Store installiert wurde, müssen Sie der App manuell vertrauen. Öffnen Sie das Gerät die Einstellungen.
  4. Klicken Allgemeines > Geräteverwaltung (Es kann auch als erscheinen Profile & Geräteverwaltung). Ihre Anwendung sollte als Unternehmensanwendung aufgeführt sein.
  5. Klicken Sie auf Ihre Bewerbung und dann Vertrauen es.
  6. Klicken Vertrauen bestätigen. Die Brightcove Engage-App ist jetzt einsatzbereit.
  7. Öffne das Anheuern App auf Ihrem Gerät.
  8. Geben Sie den Zugriffsschlüssel für das Erlebnis ein.
  9. Klicken Sie auf Enter.

Je nachdem, wie das Erlebnis konfiguriert wurde, werden Sie möglicherweise auch aufgefordert, Ihre Unternehmensdaten einzugeben, um auf die App zuzugreifen.

Installieren der nativen Android-Anwendung

Die mobile Brightcove Engage App kann auf Android-Geräten mit Android 5.0 oder höher installiert werden. Beachten Sie, dass die Schritte zum Zulassen von App-Installationen aus unbekannten Quellen je nach Android-Version variieren. Sie sollten sicherstellen, dass Ihr Gerät Installationen aus unbekannten Quellen zulässt.

Befolgen Sie diese Schritte, um die App zu installieren.

  1. Öffnen Sie die Einstellungen-App auf Ihrem Gerät.
  2. In dem persönlich Abschnitt, klicken Sie Sicherheit.
  3. Lokalisieren Unbekannte Quellen und stellen Sie sicher, dass die Ermöglichen Sie die Installation von Apps aus anderen Quellen als dem Play Store aktiviert.
  4. Öffnen Sie einen Browser und gehen Sie zu URL, in der sich die Installationsdatei befindet. Dadurch wird die App auf Ihr Gerät heruntergeladen.
  5. Bestätigen Sie die Installation durch Klicken auf INSTALLIEREN.
  6. Wenn die Installation abgeschlossen ist, klicken Sie auf ÖFFNEN um die App zu öffnen.
  7. Geben Sie den Zugriffsschlüssel für das Erlebnis ein.
  8. Klicken Sie auf Enter.

Je nachdem, wie das Erlebnis konfiguriert wurde, werden Sie möglicherweise auch aufgefordert, Ihre Unternehmensdaten einzugeben, um auf die App zuzugreifen.
