Verwalten von Videofeldern

In diesem Thema erfahren Sie, wie Sie Videofelder für ein Konto verwalten.

Mit benutzerdefinierten Metadaten können Sie Ihren Videos zusätzliche Metadatenfelder hinzufügen. Benutzerdefinierte Felder werden als Teil der Kontoeinstellungen hinzugefügt. Nachdem die benutzerdefinierten Felder hinzugefügt wurden, können den Feldern als Teil der Werte zugewiesen werden Videos Seite.

Benutzerdefinierte Felder erstellen

Gehen Sie folgendermaßen vor, um benutzerdefinierte Felder zu erstellen.

  1. Melden Sie sich bei Brightcove Engage an.
  2. Klicken Sie in der Navigationskopfzeile auf das Zahnradsymbol () and then Video Fields. Beachten Sie, dass Sie ein Kontoadministrator sein müssen, um auf das Administrationsmenü zugreifen zu können.
  3. Auf der Seite Videofelder werden Ihre benutzerdefinierten Felder und alle Standard-Videometadatenfelder angezeigt. Überprüfen Sie die Standardfelder, bevor Sie neue benutzerdefinierte Felder erstellen. Auf diese Weise können Sie vermeiden, benutzerdefinierte Felder zu erstellen, wenn bereits ein Standardfeld vorhanden ist, das die Aufgabe ausführen kann.
  4. Klicken Benutzerdefiniertes Feld hinzufügen.
  5. Geben Sie ein Anzeigename. Dies wird als Bezeichnung für das Feld auf der Seite mit den Videoeigenschaften angezeigt. Ein Interner Name wird automatisch für das Feld basierend auf dem Anzeigenamen erstellt. Der interne Name kann geändert werden, der interne Name darf jedoch keine Leerzeichen, Doppelbytezeichen oder Satzzeichen enthalten. Zum Beispiel, wenn Sie eingeben My Field Als Anzeigename lautet der interne Name my_field.
    Hinweis: Es gibt einige reservierte Wörter, die darf nicht als interne Namen für benutzerdefinierte Felder verwendet werden (der Anzeigename spielt keine Rolle). Weitere Informationen finden Sie im Abschnitt Reservierte Namen.
  6. Wählen Sie a Typ für das Feld.
    • String - Der Benutzer kann bis zu 1024 Einzelbytezeichen (oder 512 Doppelbytezeichen) alles in das Feld eingeben.
    • Aufführen - Der Benutzer wählt aus einer Liste von Werten aus. Listen können nicht mehr als 150 mögliche Werte haben und jeder Wert in der Liste darf nicht länger als 100 Zeichen sein.
  7. Geben Sie ein Beschreibung. Das Beschreibungsfeld wird als Hinweis in der Benutzeroberfläche angezeigt.
  8. Wenn Ihr Feldtyp ist Aufführen Geben Sie eine durch Kommas getrennte Werteliste ein und klicken Sie dann auf Zur Liste hinzufügen.

    Hinweise:
    1. Die Werte dürfen keine Anführungszeichen enthalten (Beispiel: Tiere, Vögel, Fische).
    2. Kein Wert darf ein Komma enthalten - alle Kommas werden als Trennzeichen für verschiedene Werte behandelt.
    Hinweis: Wenn benutzerdefinierte Feldwerte weniger als drei Zeichen enthalten, werden Sie nicht in der Lage sein, nach Videos zu suchen, die auf diesem Wert basieren. Wenn Sie beispielsweise ein boolesches benutzerdefiniertes Feld erstellen möchten, isfeatureitem , erstelle die Werteliste true und false , nicht yes und no ; Wenn Sie die letzteren Werte verwenden, werden Sie nicht Videos über die Suche finden können.
  9. Klicken Feld speichern um die Änderungen zu speichern oder klicken Sie auf Speichern und weitere hinzufügen um das Feld zu speichern und ein weiteres Feld zu erstellen.

Von Ihnen erstellte benutzerdefinierte Felder sind sofort verfügbar und wird sichtbar sein auf der Seite mit den Videoeigenschaften.

Es können bis zu 10 benutzerdefinierte Felder erstellt werden. Wenn Sie mehr als 10 Felder benötigen, klicken Sie auf Benötigen Sie weitere Felder? Link auf der Seite Videofelder, um den Brightcove-Kundensupport mit einer Anfrage zu kontaktieren. Publisher sind auf 50 benutzerdefinierte Felder beschränkt.

Listenwerte bearbeiten

Sie können einem benutzerdefinierten Feld vom Listentyp zusätzliche Werte hinzufügen. Klicken Sie auf den Link für den benutzerdefinierten Feldnamen, fügen Sie die Werte hinzu und klicken Sie dann auf Zur Liste hinzufügen. Wenn Sie fertig sind, klicken Sie auf Feld speichern.

Listenwerte löschen

Sie können vorhandene Werte aus einem benutzerdefinierten Feld des Listentyps hinzufügen und entfernen. Klicken Sie auf den Link für den benutzerdefinierten Feldnamen, klicken Sie auf das Symbol zum Löschen () associated with the value and then click Save Field.

Wenn ein Listenwert entfernt wird, behalten alle Videos, denen dieser Wert zuvor zugewiesen wurde, den Wert. Wenn jedoch ein Video mit dem gelöschten Wert bearbeitet wird, wird der Wert gelöscht und ein neuer Wert muss ausgewählt werden.

Entfernen von benutzerdefinierten Feldern aus einem Konto

Um ein benutzerdefiniertes Feld aus Ihrem Konto zu entfernen, müssen Sie Wenden Sie sich an den Brightcove-Support. Beachten Sie, dass sich das Änderungsdatum für alle Videos ändert, in denen ein Wert vorhanden war, wenn Sie vom Brightcove-Support benutzerdefinierte Felder löschen lassen.

Reservierte Namen

Es gibt einige reservierte Wörter, die darf nicht als interne Namen für benutzerdefinierte Felder verwendet werden (der Anzeigename spielt keine Rolle). Ein gebräuchlicher Name ist Kategorie. Wenn Sie einen reservierten Namen als internen Namen für ein benutzerdefiniertes Feld verwenden, werden bei der Suche in diesem Feld keine Ergebnisse zurückgegeben. Wenn Sie bereits ein benutzerdefiniertes Feld mit einem reservierten Namen haben, sollten Sie dies tun entfernen Sie es und ersetzen Sie es durch einen anderen Namen, oder wenden Sie sich an den Brightcove-Support, um weitere Informationen zu erhalten.
