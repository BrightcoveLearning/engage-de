    Benutzer verwalten

    In diesem Thema erfahren Sie, wie Sie Benutzer verwalten, die mit Brightcove Engage Zugriff haben, um Erlebnisse zu erstellen und zu veröffentlichen.
     

    Auf der Seite „Benutzerverwaltung“ werden die Benutzer verwaltet, die berechtigt sind, sich bei der Brightcove Engage-Anwendung anzumelden, um Erlebnisse zu erstellen und zu veröffentlichen. Diese Funktion wird nicht verwendet, um zu steuern, wer sich anmelden und auf von Ihnen erstellte mobile Apps und Web Experiences zugreifen kann. Beachten Sie, dass Sie ein Benutzer mit der Rolle Administrator sein müssen, um auf die Seite „Benutzerverwaltung“ zuzugreifen.

    So greifen Sie auf die User Management-Seite zu:

    1. Melden Sie sich bei Brightcove Engage™ an.
    2. Klicken Sie in der Navigationskopfzeile auf das Zahnradsymbol () und dann auf Benutzerverwaltung.

    Die Seite „Benutzerverwaltung“ wird angezeigt:

    • Nachname, Vorname
    • E-Mail-Adresse
    • Rolle - Standard oder Administrator
    • Letzte Anmeldung - Das letzte Datum und die Uhrzeit, zu der sich der Benutzer bei der Anwendung angemeldet hat.

    Auf der Benutzerverwaltungsseite können Sie:

    • Neue Benutzer erstellen
    • Bestehende Benutzer bearbeiten
    • Einen Benutzer löschen
    • Ändern Sie den Hauptbenutzer

    Wie viele Benutzer kann ich festlegen?

    Die Anzahl der Benutzer, die Sie erstellen können, hängt davon ab, welche Art von Konto Sie haben. Wenn weitere Benutzer erforderlich sind, können Sie Ihren Plan heraufstufen, sodass mehr Benutzer zulässig sind. Die Beschränkung bezieht sich nur auf die Gesamtanzahl der vorhandenen Benutzer zu einem Zeitpunkt. Wenn Sie zehn Benutzer erstellen, davon fünf löschen und drei weitere Benutzer erstellen, verfügen Sie über acht Benutzer. Auf der Seite „Benutzer verwalten“ wird angezeigt, auf wie viele Benutzer Sie Anspruch haben und wie viele zusätzliche Benutzer Sie erstellen können.

    Hinzufügen weiterer Benutzer

    Wenn weitere Benutzer erforderlich sind, können Sie Ihren Plan heraufstufen, sodass mehr Benutzer zulässig sind. Wenden Sie sich an Brightcove, um Informationen zum Hinzufügen zusätzlicher Benutzer zu erhalten.

    Neue Benutzer erstellen

    So erstellen Sie einen neuen Benutzer:

    1. Geben Sie die E-Mail-Adresse, den Vornamen und den Nachnamen des Benutzers ein.
    2. Weisen Sie dem Benutzer eine Rolle zu:
      • Administrator - Benutzer hat Zugriff auf das Administrationsmenü ()
      • Standard - Benutzer hat keinen Zugriff auf das Administrationsmenü ()
    3. Klicken Sie auf „ Benutzer hinzufügen“

    Der Benutzer wird per E-Mail darüber benachrichtigt, dass er dem Konto hinzugefügt wurde.

    Vorhandene Benutzer bearbeiten

    So bearbeiten Sie einen vorhandenen Benutzer:

    1. Klicken Sie auf den Benutzer, um ihn zu bearbeiten. Die Benutzerdetails werden oben auf der Seite angezeigt.
    2. Bearbeiten Sie die Benutzerdetails.
      Hinweis: Der Benutzer, bei dem Sie sich angemeldet haben, kann nicht bearbeitet werden.
    3. Klicken Sie auf Save, um die Änderungen zu speichern.

    Benutzer löschen

    So löschen Sie einen Benutzer:

    1. Klicken Sie auf das Löschen-Symbol () , das dem Benutzer zugeordnet ist.
    2. Klicken Sie zum Bestätigen des Löschvorgangs auf OK.

    Ändern des primären Benutzers

    Ein Benutzer in jedem Konto wird als Hauptbenutzer bezeichnet. Der primäre Benutzer hat immer die Administratorrolle und kann nur vom Brightcove Support gelöscht werden.

    Wenn Sie den Hauptbenutzer irgendwann ändern müssen, wählen Sie den neuen Hauptbenutzer aus und klicken Sie dann auf Diesen Benutzer zum primären Administrator für dieses Konto machen.

    Sie werden aufgefordert, die Änderung des primären Benutzers zu bestätigen.

    Wenn der neue Hauptbenutzer eine Rolle von Standard hat, wird ihm die Rolle Administrator zugewiesen, wenn er zum primären Benutzer gemacht wird.
