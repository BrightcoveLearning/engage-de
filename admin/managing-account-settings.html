Kontoeinstellungen verwalten

In diesem Thema erfahren Sie, wie Sie Kontoeinstellungen verwalten.

So greifen Sie auf Brightcove Engage zu™ Kontoeinstellungen, klicken Sie auf das Zahnradsymbol () in the navigation header.

Hinweis: Sie müssen ein Kontoadministrator sein, um auf das Administrationsmenü zugreifen zu können.

Das Menü Kontoeinstellungen bietet die folgenden Optionen:

  • Kontoinformationen
  • Benutzerverwaltung
  • Account-Einstellungen
  • Videofelder
  • Medien-Sharing

Kontoinformationen

Mit der Option Kontoinformationen können Sie den Kontonamen und die Postanschrift bearbeiten. Weitere Informationen finden Sie unter Kontoinformationen verwalten.

Benutzerverwaltung

Mit der Option Benutzerverwaltung können Sie die Benutzer verwalten, die zur Verwendung des Brightcove Engage-Kontos berechtigt sind. Weitere Informationen zum Verwalten von Benutzern finden Sie unter Benutzer verwalten.

Account-Einstellungen

Mit der Option Kontoeinstellungen können Sie die Standardzeitzone für die Berichterstellung für das Konto festlegen. Die Berichtszeitzone wird verwendet, um Tagesgrenzen für die Berichterstellung von Daten zu berechnen. Die Änderung der Zeitzoneneinstellung für die Berichterstellung wirkt sich nur auf die künftigen Daten aus, und Änderungen werden nicht rückwirkend angewendet. Wenn Sie diese Einstellung ändern, sehen Sie möglicherweise eine flache Stelle oder einen Anstieg Ihrer Analysedaten, wenn Sie sich Tagesgrenzen ansehen, an denen die Änderung angewendet wurde. Aktualisierungen dieser Einstellung werden möglicherweise nicht sofort wirksam, und Analysedaten beziehen sich möglicherweise weiterhin auf die alte Einstellung, bis das System die Änderungen verarbeitet.

Videofelder

Mit der Option Videofelder können Sie benutzerdefinierte Metadatenfelder für Ihr Konto anzeigen, erstellen und ändern. Weitere Informationen zum Konfigurieren von Videofeldern finden Sie unter Verwalten von Videofeldern.

Medien-Sharing

Mit der Option Medienfreigabe können Sie Videos einmal hochladen und die Videos dann für andere Brightcove-Konten freigeben. Weitere Informationen zum Teilen von Medien finden Sie unter Verwalten der Einstellungen für die Medienfreigabe.
